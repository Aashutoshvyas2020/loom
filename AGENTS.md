# Loom

Loom exposes eight authenticated MCP tools: terminal, agents, read, write, edit, skills, memory, and a dedicated browser.

- Keep configured roots narrow.
- Treat terminal access as full local-user execution, not a sandbox.
- Keep tool results bounded and schemas flat.
- Preserve owner OAuth, HTTPS public origins, process-group cleanup, and private-network browser blocking.
- Prefer deletion, standard library, focused tests, and the minimal file map in `CHANGELOG.md`.

# Configuration

Run `loom init`, then `loom launch`. Files live at `~/.loom/config.json` and `~/.loom/auth.json`.

| Variable | Default |
| --- | --- |
| `HOST` | `127.0.0.1` |
| `PORT` | `7676` |
| `LOOM_ALLOWED_ROOTS` | current directory |
| `LOOM_PUBLIC_BASE_URL` | loopback URL; public values must use HTTPS |
| `LOOM_ALLOWED_HOSTS` | local and public URL hosts |
| `LOOM_OAUTH_OWNER_TOKEN` | generated by `loom init` |
| `LOOM_STATE_DIR` | `~/.local/share/loom` |
| `LOOM_SKILL_PATHS` | additional comma-separated skill roots |
| `LOOM_LOG_LEVEL` | `info` |
| `LOOM_LOG_FORMAT` | `json` |
| `LOOM_LOG_REQUESTS` | `1` |
| `LOOM_TRUST_PROXY` | `0`; `loom launch` enables loopback trust |

OAuth TTL overrides: `LOOM_OAUTH_ACCESS_TOKEN_TTL_SECONDS`, `LOOM_OAUTH_REFRESH_TOKEN_TTL_SECONDS`, `LOOM_OAUTH_SCOPES`, and `LOOM_OAUTH_ALLOWED_REDIRECT_HOSTS`.
